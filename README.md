# Voice-Traffic-Analyzer
Cisco CUBE cihazları üzerinden geçen aktif ses trafiğini gerçek zamanlı analiz eden Python aracı. Karmaşık çağrı bacaklarını (Call Legs) otomatik eşleştirir, dahili ve cep numaralarını ayıklar, bağlantı durumlarını (Active/Connecting) raporlar. Örnek çıktıya README dosyası üzerinden bakabilirsiniz.
